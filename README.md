# Stone Circle Marketing Website

Marketing website for Stone Circle - time zone awareness for distributed teams.

## Files

- `index.html` - Main landing page
- `support.html` - Support conservation page
- `styles.css` - Website styling (stone gray theme)
- `script.js` - Mobile menu and interactions
- `logo.png` - Stone Circle logo

## Deployment

This is a static website. Deploy to any static hosting:

- Netlify
- GitHub Pages
- Vercel
- Cloudflare Pages

No build step required - just upload files.

## Local Development

```bash
# Simple server
python3 -m http.server 8000

# Or use any static file server
npx http-server
```

Visit `http://localhost:8000`

## Design

- Calm, professional aesthetic
- Stone gray color palette matching the app
- Grounded, earthy theme
- Privacy-first messaging
- Conservation-focused

## Structure

- **Hero** - "Time zone awareness for distributed teams"
- **Problem/Solution** - Coordination challenges
- **Features** - Meeting planner, travel tracking, etc.
- **Philosophy** - Why "Stone Circle"
- **Privacy** - Local-only data, no tracking
- **Our Story** - Built for conservation teams
- **Get Started** - PWA installation
- **How to Install** - Platform-specific instructions
- **Contact** - Embedded form
- **Support** - Link to Amphibian Survival Alliance
